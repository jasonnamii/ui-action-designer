## §1. 8단계 게이트 체크리스트

| 단계 | 게이트 | PASS 기준 | 실패 시 |
|------|--------|-----------|---------|
| ① 입력 | 재현도 판정 | 소스·MCP 연결·폴백 안내 | 안내 누락 = 재진입 |
| ② DS 추출 | 토큰 게이트 | 색상·타이포·간격·radius·shadow 테이블 + **형 승인** | 승인 없이 ③ = FAIL |
| ③ 분석 | 경로 판정 | A/B/C 중 1개 확정 | 미확정 = STOP·형 확인 |
| ④ 추출 | 구조·인터랙션 | 레이아웃·컴포넌트·계층 + 탭/스와이프/롱프레스 식별 | 누락 = ④ 재실행 |
| ⑤ SHE | 4축 + 타이밍 + **형 승인** | Before/After·등급 근거 | 승인 없이 ⑤-b = FAIL |
| ⑤-b UX_GATE | 15원리 중 **12개+ PASS** | Nielsen 10 + Norman 5 | 11↓ = SHE L3 재설계 or ④ 롤백 |
| ⑥ 지식 유입 | Action·Task 특정 사례 | 범용 원칙 ✗ | 혼입 감지 = 재수집 |
| ⑦ 제안 | 4방향 차별화 | 2+ 정보계층·흐름 차별 | 동일 구조 = 재드래프트 |
| ⑧ 산출 | PRD 정합성 매트릭스 | PRD 존재 시 §4 | 스킵 = FAIL |
